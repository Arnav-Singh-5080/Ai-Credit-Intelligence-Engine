# Security Policy

## Supported Versions

This project is under active development. Security updates are currently applied to the latest version only.

| Version | Supported |
| ------- | --------- |
| Latest  | ✅        |

---

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

You can report vulnerabilities by:
- Opening a GitHub issue
- Contacting the maintainers (if contact details are available)

Please avoid publicly disclosing security issues until they have been addressed.

----

## Security Best Practices

- Keep dependencies up to date
- Avoid exposing sensitive information
- Validate all user inputs
- Follow secure coding practices
